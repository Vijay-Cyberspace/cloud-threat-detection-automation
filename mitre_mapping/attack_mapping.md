# MITRE ATT&CK Mapping

This project aligns cloud detection logic with MITRE ATT&CK techniques.

## T1078 – Valid Accounts
Detection of compromised IAM credentials and abuse of legitimate cloud accounts.

## T1098 – Account Manipulation
Unauthorized policy attachment, privilege escalation, and IAM role abuse.

## T1552 – Unsecured Credentials
Excessive access key creation and exposed cloud credentials.

## T1071 – Command & Control
Outbound communication with malicious IP addresses detected via enrichment APIs.
